🏥 Hospital Management Database System

Our project is a Hospital Management Database System designed to manage patients, doctors, appointments, rooms, and medical records in an efficient and organized way.

💾 The database was implemented using SQL, and it includes the use of important transaction management concepts such as:
- COMMIT   
- ROLLBACK  
- SAVEPOINT   

These ensure data consistency, safety, and reliability during all database operations.

🧠 The system also applies several core database concepts, including:
- Primary and Foreign Keys   
- Relationships between tables  
- Constraints   
- Joins  
- Views   
- Stored Procedures   
- Transactions
